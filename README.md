# String Calculator TDD Kata

Implementation for Incubyte assessment using TDD in Ruby. Uses RVM with gemset `incubyte_kata` for isolation. No Rails requiredâ€”pure Ruby with RSpec.

## Setup
- Install RVM: `curl -sSL https://get.rvm.io | bash -s stable`
- Install Ruby: `rvm install 3.2.2`
- Create gemset: `rvm use 3.2.2@incubyte_kata --create`
- Install gems: `bundle install`
- Run tests: `./bin/run_tests.sh`
